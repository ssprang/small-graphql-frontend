# Small graphql frontend
This repo contains an example api using:
* Vite
* Vue 3
* Vuetify (Beta)
* TypeScript
* GraphQL

Schema based on [small-graphql-api](https://github.com/ssprang/small-graphql-api).

## How to run it
```sh
npm install
npm run dev
```

## Generate types
```sh
npm run codegen --watch
```
